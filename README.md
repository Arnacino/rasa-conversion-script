# Rasa Conversion Script

A script to convert a Rasa chatbot from version 1.x to version 3.x.

## Requirements
- Python
- A virtual environment with **Rasa 2.x** installed
- An existing Rasa chatbot

## Important Notes
- The script only works with Rasa chatbots that have a `data` folder containing `rules.md` and `stories.md`.

## Usage
1. Download the script.
2. Activate your virtual environment with Rasa 2.x in a terminal.
3. Update the `SOURCE_DIR` variable in the script to point to the root folder of your chatbot.
4. Run the script.
